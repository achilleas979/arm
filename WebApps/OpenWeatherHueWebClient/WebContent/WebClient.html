WebClient - SmartPhilipsHUELight && OpenWeather

WebClient for interacting with SmartPhilipsHUELight && OpenWeather.

This simple web app allows calling OpenWeather ARM plugin REST API and if the city temperature returned is >30celsius it turns on the Hue light else turns it off.
